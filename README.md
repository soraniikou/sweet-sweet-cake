🍓クマさんが　あなたの思いを受け取る🐻

伝えたい言葉が言えない時に
クマさんを5回タップして　言葉を書いてみてください
消去できます
保存して見返すこともできます

また,
7日後に確認ボタン を選択して、7日後に同じ気もちかどうか再度　気持ちを確認できます。夜空🌌に言葉を預けます

ご自由にお使いください
アプリ体験
https://sweet-sweet-cake.vercel.app
